This site is built with [Astro](https://astro.build) and deploys as fully static
HTML. Articles and projects are authored as Markdown files using Astro's
type-safe **content collections**, so publishing is as simple as committing a
new `.md` file.

## Highlights

- Zero client-side JavaScript by default — pages are plain HTML + CSS.
- Light and dark themes via `prefers-color-scheme`.
- Automatic sitemap generation.
- Host-agnostic build — works on GitHub Pages, Vercel, or Netlify.

## Stack

- **Astro** for routing, content collections, and static rendering.
- **TypeScript** for the schema definitions.
- Plain CSS, no UI framework.
